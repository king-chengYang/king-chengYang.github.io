选择排序

时间复杂度：$O(n^2)$

方法：依次找到数组中第一小，第二小，第三小…的数，并放置在对应的位置上

每次排序后的结果：每次将未排序的数中的最小值排到正确的位置上

public int[] selectionSort(int[] num){
    int len = num.length;

    for (int i=0;i<len;i++){
        int position = i;
        int min = num[i];
        //找到未排序的数中最小的数
        for (int j=i+1;j<len;j++){
            if (num[j]<min){
                min = num[j];
                position = j;
            }
        }
        //将找到的数放置在对应位置
        int mid = num[i];
        num[i] = num[position];
        num[position] = mid;
    }
    return num;
}

冒泡排序

时间复杂度：$O(n^2)$

方法：当前的数与后一个数比较，如果比后面小，互换位置继续比较；如果比后面大，则用后面的代替该数继续比较

每次排序后的结果：每次将未排序的数中的最大数放置在对应的位置

  public int[] bubbleSort(int[] num){
      int len = num.length;

      for (int i=0;i<len;i++){
          int max = num[0]; //每次选定第一个数，每一轮后第（len-i）排好序
          for (int j=1;j<len-i;j++){
              if (num[j]>max){ //如果后面小，不用变动数据位置，直接将下一个设置为最大
                  max = num[j];
              }else{ //如果比后面大，则交换位置
                  num[j-1] =num[j];
                  num[j] = max;
              }
          }
      }
      return num;
  }

插入排序

时间复杂度：$ O(n^2) $

方法：选定一个数，向前比较，将它插入到比它大的数之前

每次排序后的结果：放置好之后的数（包括该数）之前有序，即形成局部有序

public int[] insertSort(int[] num){
    int len = num.length;
    for(int i=0;i<len;i++){
        int sortNum = num[i]; //排第i个数
        for (int j=i-1;j>=0;j--){
            if(num[j]>sortNum){ // 将比他大的数换一下位置
                num[j+1]= num[j];
                num[j] = sortNum;
            }
        }
    }
    return num;
}

快速排序

时间复杂度：$O(n * logn)$

方法：选定一个数，然后从前向后指针找比该数大的数，从后向前指针找比该数小的数，再交换这两个数，直到从前向后的坐标大于从后向前的坐标，之后再将选定的数与从前向后的最后指向的数交换位置，迭代进行

每次排序后的结果：将选定的数排在正确的位置，左边全是比它小的数，右边全是比它大的数

//num为待排序的数组，left为左边界，right为右边界，每次选定数组的最左边的元素为待排元素
public int[] quickSort(int[] num, int left, int right){
    int sortNum = num[left];
    int i=left+1;
    int j=right;
    while (true){
        //找比选定的数大的数
        while (i<=right && num[i]<sortNum)
            i++;

        //找比选定的数小的数
        while (j>left && num[j]>sortNum)
            j--;

        //交换
        if (i<j){
            int mid = num[j];
            num[j] = num[i];
            num[i] = mid;
        }else
            break;
    }
    //交换选定元素sortNum和最后left指定的数
    num[left] = num[j];
    num[j] = sortNum;

    //分别对前半部分，后半部分排序，迭代
    if (left<j-1)
        quickSort(num,left,j-1);
    if (right>j+1)
        quickSort(num,j+1,right);
    return num;
}

归并排序

时间复杂度：$O(n*logn)$

方法：自低向上，不断合并每个小有序数据，最后形成最终有序

每次排序后的结果：每个小有序数据合并成大的有序数列

 public int[] mergerSort(int[] num,int left,int mid,int right){
     //只有一个元素时直接返回
     if (right == left){
         int[] answer = {num[right]};
         return answer;
     }

     int[] answer = new int[(right-left+1)];
     int[] leftNum;
     int[] rightNum;

     int leftMid = (int)((left+mid)/2);
     int rightMid = (int)((right+mid)/2);

     leftNum = mergerSort(num,left,leftMid,mid);
     rightNum = mergerSort(num,mid+1,rightMid,right);

     //合并
     int leftLen = leftNum.length;
     int rightLen = rightNum.length;

     int position = 0;
     int i=0,j=0;
     for(;i<leftLen && j<rightLen;){
         if (leftNum[i]<rightNum[j]){
             answer[position++] = leftNum[i++];
             continue;
         }
         if (leftNum[i]>rightNum[j]){
             answer[position++] = rightNum[j++];
             continue;
         }
     }

     while (i<leftLen){
         answer[position++] = leftNum[i++];
     }
     while (j<rightLen){
         answer[position++] = rightNum[j++];
     }
     return answer;
 }
