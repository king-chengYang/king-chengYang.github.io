Hbase Shell

HBase提供了一个非常方便的命令行交互工具HBase Shell。

以下表（s_behavior）为例

数据定义语言

数据定义语言（Data Definition Language，DDL），包括数据库表的创建，修改等语句。

创建表

create <table>, <NAME => 列族>

create 's_behavior', {NAME => 'pc'},{NAME => 'ph'}

查看所有表

list

查看建表

describe 's_behavior'

表的属性解释

  • DATA_BLOCK_ENCODING：数据块编码。用类似于压缩算法的编码形式来节省存储空间。
  • BLOOMFILTER：布隆过滤器。数据查询Scan操作的时候用来排除待扫描的StoreFile文件。
  • REPLICATION_SCOPE：集群间数据赋值开关。1表示开启赋值。
  • VERSIONS：HBase对表的数据行可以保留多个数据版本，以时间戳来区分。
  • COMPRESSION：压缩方式。HBase提供多种压缩方式用来在数据存储到磁盘之前压缩以减少存储空间。
  • TTL： 数据的有效时长，超过TTL的数据在主压缩时会被删除。
  • KEEP_DELETED_CELLS：保留了删除的数据，当主压缩后删除的数据才会被清理，如果开启集群间复制，则必须为true，否则会导致数据复制失败
  • BLOCKSIZE：HBase读取数据的最小单元，默认为64KB。

修改表

修改表需要先将表下线，然后执行修改命令，再上线

disable 's_behavior' #下线
alter 's_behavior', {NAME => "cf", REPLICATION_SCOPE => "1", KEEP_DELETED_CELLS => "TRUE"} #改变表
enable 's_behavior' #上线

数据操作语言

数据操作语言（Data Manipulation Language，DML），包括数据的修改，查询，删除等语句。

PUT

put命令用来插入一行数据到HBase表

put <table>, <rowkey>, <列族:列标识符>, <值>

#插入pc值
put 's_behavior', '12345_151692489001_1', 'pc:v', '1001'

#插入ph值
put 's_behavior', '12345_151692489001_1', 'ph:o', '1001'

GET

get命令用来根据行键获取HBase表的一条记录

get <table>, <rowkey>

#获取一条数据
get 's_behavior', '12345_151692489001_1'

# 获取一条数据某时刻的镜像
get 's_behavior', '12345_151692489001_1', {TIMESTAMP => '151692489001'}

#获取多个版本的数据(get命令支持获取数据的多个版本，建表是指定VERSIONS属性即可)
alter 's_behavior', NAME=>'pc', VERSIONS=>3 #改表
put 's_behavior', '12345_151692489001_1', 'pc:v', '1002' #放一个新数据，之前已经有该键
get 's_behavior', '12345_151692489001_1', {COLUMN => 'pc:v', VERSIONS => 2}

Scan

Scan命令用来扫描表的数据，Scan可以有多种用法

scan <table>

#扫描表
scan 's_behavior'

#获取时间区间内数据
scan 's_behavior', {TIMERANGE => [151423996739,151513996739]}

#获取多版本数据
scan 's_behavior', {VERSIONS => 2}

#获取用户(ID=12345)前5行数据
scan 's_behavior', {FILTER => "PrefixFilter('12345_')", COLUMNS => ['pc'], LIMIT=>5}

#获取用户(ID=12345)某个时间区间的PC端行为数据
scan 's_behavior', {STARTROW => '12345_151423996739',STOPROW => '12345_151513996739', COLUMNS => ['pc']}

#获取用户对商品(ID=1001)的行为数据
scan 's_behavior', FILTER => "ValueFilter(=,'binary:1001')"

#获用户对商品(ID包含1002)的行为数据
scan 's_behavior', FILTER => "ValueFilter(=,'substring:1002')"

Delete

delete用来删除某列数据

delete <table>, <roekey>, <列族:列标识符>, [<time stamp>]

delete 's_behavior', '12345_151692489001_1', 'ph:o'

DeleteAll

deleteall删除整行数据

deleteall 's_behavior', '12345_151692489001_1'

truncate

truncate删除整个表

truncate 's_behavior'
